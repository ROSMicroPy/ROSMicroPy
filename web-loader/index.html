ROSMicroPy firmware loader

Install ROSMicroPy Core

Flash a ROS 2 compatible MicroPython node directly from your browser. The loader uses ESP Web Tools and selects the right firmware for generic ESP32 and ESP32-S3 Core boards.

Web install

Connect the board over USB, choose the serial port, and install the matching ROSMicroPy Core image.

Web Serial is not available in this browser. Use Chrome, Edge, or another Chromium-based desktop browser.

Included builds

RMP Core Generic

Uses rmp_core_generic for generic ESP32 boards.

RMP Core S3

Uses rmp_core_s3 for ESP32-S3 boards.

Before flashing

  1. Use a data-capable USB cable.
  2. Close serial monitors or tools already connected to the board.
  3. Back up any MicroPython files you want to keep.
